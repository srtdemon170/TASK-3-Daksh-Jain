# Random Password Generator

## Project Overview

Random Password Generator is a Python console application that creates secure and unpredictable passwords based on the length provided by the user. The program combines uppercase letters, lowercase letters, digits, and special characters to generate a strong password.

This project demonstrates the use of Python libraries, string manipulation, randomization, loops, and input validation.

---

## Features

- User chooses the password length.
- Generates a strong random password.
- Includes:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Special characters
- Validates user input.
- Easy-to-use console interface.

## How to Run

1. Install Python 3.
2. Save the file as `random_password_generator.py`.
3. Open Terminal or Command Prompt.
4. Run the following command:

```bash
python random_password_generator.py
```

## Project Workflow

1. Ask the user for the desired password length.
2. Validate the input.
3. Select random uppercase letters, lowercase letters, digits, and symbols.
4. Shuffle the characters.
5. Display the generated secure password.
